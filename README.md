#  Система управління студентською бібліотекою

Це десктопний додаток для автоматизації роботи бібліотекаря: він дозволяє вести облік літератури, реєструвати читачів (студентів) та контролювати видачу книг із перевіркою лімітів.

##  Технологічний стек
* **Мова:** Python 3.10+
* **Інтерфейс (GUI):** PyQt5
* **База даних:** SQLite3 (вбудована, не потребує окремого сервера)
* **Тестування:** `unittest` + `pytest`

##  Основний функціонал
* **Каталог літератури:** Додавання, редагування, пошук та списання книг.
* **Облік студентів:** Реєстрація читачів, пошук за групою чи ПІБ.
* **Контроль видачі:** * Реєстрація видачі книги конкретному студенту.
  * Автоматичне блокування видачі, якщо студент перевищив ліміт (максимум 4 книги на руки).
  * Блокування видачі, якщо вільних примірників не залишилося.
* **Моніторинг:** Перегляд списку активних боргів та прострочених книг.

##  Архітектура та Патерни
Проєкт написано з дотриманням принципів ООП та SOLID. В основі лежить архітектурний шаблон **MVC (Model-View-Controller)**.

Також у коді реалізовані GoF-патерни:
* **Repository:** Всі SQL-запити винесені в окремі класи-репозиторії (`repositories/`), щоб відв'язати логіку від конкретної бази даних.
* **Facade:** Центральний клас (`services/facade.py`), який керує всіма складними процесами (наприклад, перевіркою лімітів перед видачею).
* **Observer (Шина подій):** Спеціальний механізм (`patterns/gof.py`), який дозволяє вкладкам інтерфейсу автоматично оновлюватися, коли в базі відбуваються зміни, без жорсткого зв'язування коду.
* **Singleton:** Забезпечує єдине підключення до файлу БД.
